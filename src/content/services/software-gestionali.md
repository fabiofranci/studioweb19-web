Progettazione e sviluppo di software gestionali web orientati a workflow operativi reali.

- Laravel
- MySQL
- FilamentPHP
- API REST
- Queue jobs

## Ambiti applicativi

- piattaforme SaaS;
- workflow documentali;
- sistemi operativi verticali;
- dashboard operative;
- reportistica.
